# Infinity New Tab Extension 🚀

一个功能丰富、界面精美的 Chrome 新标签页扩展，让你的每一次打开新标签都成为愉悦的体验。

## ✨ 主要功能

### 🖼️ 壁纸系统
- **本地上传**：支持上传任意图片作为背景
- **精选壁纸**：8张高质量预设壁纸（来自 Unsplash）
- **二次元源**：可一键从二次元壁纸站拉取随机背景
- **自定义调节**：
  - 可调节模糊度（0-10px）
  - 可调节暗度覆盖层（0-80%）
- **一键重置**：恢复默认渐变背景

### 🔍 智能搜索
- **多引擎支持**：Google、Bing、百度、DuckDuckGo
- **快捷键**：按 `/` 立即聚焦搜索框
- **引擎切换**：在设置中一键切换
- **毛玻璃效果**：美观的搜索框设计

### 📊 状态概览
- **活动监测**：正在播放的媒体标签、进行中的下载
- **系统信息**：CPU 线程数、设备内存、可用时显示电池状态

### 🧭 最近常访问
- **常用网站聚合**：自动提取近期常访问的网站，横向滑动浏览

### 🔖 增强的书签管理
- **文件夹分类**：更好地组织你的书签
- **自动获取图标**：从 Google Favicon Service 自动获取网站图标
- **快速访问**：点击书签在新标签页打开
- **悬停删除**：悬停显示删除按钮
- **启动台视图**：像 macOS Launchpad 一样的文件夹卡片浏览

### ⚙️ 强大的设置面板
侧边滑出式设置面板，包含4个选项卡：

#### 1. 外观
- 时钟格式（12/24小时制）
- 搜索引擎选择

#### 2. 壁纸
- 随机壁纸
- 上传本地图片
- 重置默认背景
- 模糊度调节
- 暗度调节

#### 3. 布局
显示/隐藏组件开关：
- 时钟
- 搜索框
- 书签
- 状态概览
- 最近常访问

#### 4. 数据
- **导出数据**：将所有设置和数据导出为 JSON
- **导入数据**：从 JSON 文件恢复数据
- **重置设置**：一键恢复默认设置

### 💾 数据同步
使用 `chrome.storage.sync` 实现：
- 自动保存所有设置
- 跨设备同步（同一 Chrome 账号）
- 实时更新

## 🎨 设计特色

- **现代美学**：渐变背景、毛玻璃效果
- **流畅动画**：所有交互都有过渡动画
- **响应式设计**：在各种屏幕尺寸下都表现出色
- **直观操作**：悬停显示控制按钮，减少视觉杂乱

## 📦 安装方法

### 从源码安装（开发者模式）

1. **下载代码**
   ```bash
   cd /Users/kejincai/dev/infinity-newtab-extension
   ```

2. **打开 Chrome 扩展页面**
   - 在地址栏输入 `chrome://extensions`
   - 或通过菜单：更多工具 → 扩展程序

3. **启用开发者模式**
   - 点击页面右上角的"开发者模式"开关

4. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择项目目录
   - 点击"选择"

5. **开始使用**
   - 打开新标签页（Cmd+T 或 Ctrl+T）
   - 享受全新的标签页体验！

## 🔧 技术栈

- **原生技术**：HTML、CSS、JavaScript (ES6+)
- **Chrome API**：
  - `chrome.storage.sync`：数据持久化和同步
  - `chrome_url_overrides`：新标签页覆盖
- **架构模式**：模块化单例模式
- **设计**：Glassmorphism（毛玻璃）风格

## 💡 使用技巧

1. **快速搜索**：按 `/` 键立即聚焦搜索框

## 🐛 已知问题

1. **数据限制**：`chrome.storage.sync` 有配额限制（总大小 102,400 bytes）
2. **图标依赖**：书签图标依赖 Google Favicon Service（某些地区可能需要代理）
3. **浏览器兼容**：仅支持 Chromium 内核浏览器（Chrome、Edge 等）
